FUCNCTION 1 USER_LOGIN
This function will attempt to log a user in. If the user_id exists the user will be logged in and their id will be saved to the user session.
Else the user will be prompted to create a new id. After successful creation their user id will be saved to the user_session, and the new user will be added to the USER database

Inputs: 
user_id | int

Outputs:
updated user session | dict


create log_in_user function
prompt user for user_id
While the user enters the id
    check that the user id is an integer using an int check | int(user_input)
        if true, break out of loop
        if false ask the user to enter a valid input
    While ...  
        check that the id exists in the USER database with an if in statement | if x in dict
            if id exists, save user_id to user_session | key:active_user, value: user_id
                exit out of loop
            if id does not exist,
                - tell user the id does not exist
                - give user a yes or no option to create a user_id
                    validate yes or no response by ensure the input is a string and convert to lowercase | str(),  .strip().lower()
                        if yes prompt user to enter a user_id as a digit
                            validate response by ensuring user input is a digit
                            if valid 
                                udate USER database with new id | USERS[user_id] = {}
                                save user_id to user_session | key:active_user, value: user_id
                                exit out of loop
                            if not valid loop until user enters a valid input
                        if no break out of loop and ask user for a valid id
                        if invalid, stay in loop until user inputs a yes or no response
