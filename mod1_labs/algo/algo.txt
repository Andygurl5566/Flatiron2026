This system should be able to:
- set new savings goals based on specific users 
- set dates to achieve said goals
- update the savings progress toward the goals
- notify the user when a goal date is approaching and the savings has not been met

Create an empty dict for USERS database. It will look like this:

USERS = {
        USER_ID:{
            GOAL_NAME:{
                TARGET_DATE
                TARGET_AMOUNT
                AMOUNT_SAVED
            }
        }
    }


HELPER FUNCTIONS:

USER_LOGIN HELPER FUNCTION
Function login_user() output: USER_ID
    loop until valid entry
    prompt user for USER_ID
        ensure the USER_ID exists in the system by checking the USERS database
            if the USER_ID exists, break loop
            else prompt user to create an id or try to enter their id again
    return the USER_ID

Function amount_is_valid(AMOUNT) output: BOOL
    if AMOUNT cannot be converted to a float return False
    if AMOUNT <= 0 return False
    else return True

Function date_is_valid(DATE) output:BOOL
    if DATE can be a datetime value return True
    else return False


MAIN FUNCTIONALITY:

Retrieve USER_ID from login_user function

Present user with menu options [CREATE GOAL, UPDATE GOAL]
    if CREATE GOAL run create create_saving_goal function
    if UPDATE GOAL run update_saving_progress function
    else ask for a valid input

    using USER_ID, retrieve goals
    run the saving_goal_notification on each goal to see if anything is due soon


Function create_saving_goal(USER_ID) | output: stores dictionary with GOAL_NAME as the key and TARGET_DATE, TARGET_AMOUNT, and AMOUNT_SAVED as values
    loop until valid entry
        prompt user for a GOAL_NAME
            ensure the goal name does not already exist in the USERS database, and is not empty
            if the goal name is valid, store the value in GOAL_NAME variable and break loop
            else loop until valid
    loop until valid entry
        prompt user for a TARGET_AMOUNT and validate using amount_is_valid function
            if valid store to TARGET_AMOUNT variable and break loop
            else loop until valid
    loop until valid entry
        prompt user for a TARGET_DATE in YYYY-MM-DD format and validate date with date_is_valid helper function
            if valid input, store value to TARGET_DATE variable and break loop
            else loop until valid
    create a goal dictionary
    GOAL_NAME : {TARGET_DATE, TARGET_AMOUNT, AMOUNT_SAVED: 0} <-- amount saved defaults to 0 when first created
    store goal dictionary under USERS[USER_ID][GOAL_NAME]

    notify user change was successful
    Return to menu


Function update_saving_progress(USER_ID, GOAL_NAME) | updated AMOUNT_SAVED in USER database
    Retrieve the AMOUNT_SAVED from the database with USER_ID and GOAL_NAME
    loop until valid entry
        prompt user for AMOUNT_TO_ADD and validate using amount_is_valid function
                if valid update AMOUNT_SAVED by adding AMOUNT_TO_ADD
                    and break loop
                else loop until valid

    notify user change was successful
    Return to menu
        

Function saving_goal_notification(USER_ID, GOAL_NAME) 
    import datetime 
    capture the current date

    retrieve the TARGET_DATE and TARGET_AMOUNT, AMOUNT_SAVED using the USER_ID and GOAL_NAME

    create a days_left variable from the target_date - datetime.date.today()
    create a goal_is_met = AMOUNT_SAVED >= TARGET_AMOUNT

    if days_left < x amount of days and goal_is_met is False. 
        return a warning formatted string
    else 
        return nothing
